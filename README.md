> # ForexWar_TN
> 
> A forex (foreign exchange) market simulation.

An API-based solution that allows the following :
1. Profile Creation
2. Profile Checking
3. Orders Creation
4. Orders Changing
5. Orders Checking
6. Deleting Profile
